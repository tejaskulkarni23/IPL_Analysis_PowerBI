# IPL_Analysis_PowerBI
Developed an interactive IPL Analysis Dashboard in Power BI using 18+ seasons of IPL data (2008–2026) to analyze team performance, player statistics, match outcomes, and tournament trends. The dashboard provides dynamic insights through KPIs, advanced DAX calculations, drill-through reports, and interactive visualizations.
🏏 IPL Analysis Dashboard | Power BI
📌 Project Overview

This Power BI project analyzes the complete history of the Indian Premier League (IPL) from 2008–2026. The dashboard provides comprehensive insights into team performance, player statistics, match outcomes, venue analysis, and season trends through interactive visualizations and KPIs.

🎯 Objectives
Analyze overall IPL performance across seasons.
Identify top-performing teams and players.
Compare batting and bowling statistics.
Explore venue-wise and season-wise trends.
Visualize head-to-head team records and win percentages.
